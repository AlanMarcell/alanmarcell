# Setup Git

## Auto Completion

```
wget https://github.com/git/git/blob/master/contrib/completion/git-completion.bash --quiet --show-progress -O ~/git-completion.bash

echo -e "\nsource ~/git-completion.bash" >> ~/.bash_profile
```
