How to create a Docker Alpine and Nodejs envirioment

1) Install docker

Install docker on Debian

2) Install Alpine

Install Alpine Download Alpine docker pull alpine Get inside the container docker run -i -t alpine /bin/sh Detaching from the container without stopping Ctrl-P Ctrl-Q Check the docker container is still running docker ps -a

3) Install Nodejs

apk update apk add nodejs Testing Node node -v

4) Install mongodb

echo 'http://dl-3.alpinelinux.org/alpine/edge/testing' >> /etc/apk/repositories apk upgrade --update apk add mongodb
